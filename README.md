## How to play:
Explore the dungeon you accidentally created and fight powerful creatures!  
Then collect their essence to **summon** them as your allies!
  
## Controls:
- **WASD** >> Move
- **Mouse** >> Aim
- **LMB** >> Shoot
- **R** >> Return to Witch's Hut
- **Esc** >> Quit application
